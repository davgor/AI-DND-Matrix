# API key settings UI + secure save path

## Description
Enable users to enter/update API keys from Settings with secure handling and redacted display rules.

## Acceptance Criteria
- [ ] Settings form supports entering/updating API key values for API-key provider mode
- [ ] Key fields are masked/redacted in UI and never displayed in plain text after save
- [ ] Renderer does not receive raw stored secrets beyond explicit write/test operations
- [ ] Save path uses existing secure main-process config/secrets boundaries
- [ ] Unit tests verify redaction and secure handling behavior
