# Startup smoke test (dev + packaged)

## Description
Validate startup loading behavior end-to-end in both development and packaged app contexts.

## Acceptance Criteria
- [ ] Dev-mode smoke confirms loading screen appears, DB stage runs, LLM stage runs, then handoff to app shell
- [ ] Packaged-mode smoke confirms same behavior for downloaded users
- [ ] At least one expected-failure smoke (runtime missing/unreachable) shows failure UX and retry path
- [x] Smoke runbook documents setup, steps, and observed outcomes
