# llama local enable + runtime/model setup flow

## Description
Provide a guided settings flow to enable local llama mode, including runtime acquisition/setup and model path configuration.

## Acceptance Criteria
- [ ] Settings include toggle/selection to enable local llama provider mode
- [ ] Flow supports configuring runtime executable path and model path
- [ ] Flow supports guided download/setup path (or explicit install instructions) for llama runtime when missing
- [ ] Runtime/model validity checks run before allowing apply/ready
- [ ] Failure states (missing runtime/model, invalid path, launch failure) are actionable and recoverable
