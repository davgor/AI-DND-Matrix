# Provider mode selector + schema

## Description
Add provider mode selection and typed settings schema so users can switch between API-key provider, local llama provider, and Player2 endpoint mode.

## Acceptance Criteria
- [ ] Settings include provider mode selector with at least: Claude/API-key, llama local, Player2 endpoint
- [ ] Provider-specific field groups show/hide correctly based on selected mode
- [ ] Typed settings schema validates required fields per mode
- [ ] Mode switch does not silently discard previously saved provider-specific values
