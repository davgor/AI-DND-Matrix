# Startup orchestrator (main process + IPC)

## Description
Implement a startup orchestrator in main process that runs boot stages in order and streams progress events to renderer.

## Acceptance Criteria
- [ ] Main-process startup orchestrator executes defined boot stages in deterministic order
- [ ] Renderer receives progress events through preload-safe IPC APIs (no direct node access in renderer)
- [ ] Orchestrator can be awaited by app shell to block ready-state until boot completes
- [ ] Unit tests verify normal path and stage failure propagation
