# M003.11 - Compatibility and save integrity under mod churn

## Description
Protect campaign continuity when mods are updated, disabled, or removed.

## Acceptance Criteria
- [ ] Existing save references to mod content resolve safely or degrade gracefully
- [ ] Disabled/removed mod entities surface clear fallback behavior in UI/runtime
- [ ] Integrity checks detect broken references introduced by mod churn
- [ ] Recovery tools or guidance provided for incompatible mod changes
