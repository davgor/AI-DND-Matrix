# m001.11 Moderation/safety and prompt sanitization policy

## Description
Define and enforce safety constraints for generated visual content and prompt inputs.

## Acceptance Criteria
- [ ] Prompt sanitization rules exist for user and model-generated image prompts
- [ ] Safety policy defines blocked/allowed classes and fallback behavior
- [ ] Unsafe generation attempts are handled gracefully and logged with redaction
- [ ] Tests cover representative safe/unsafe prompt handling paths
