# m001.4 Player-view background refresh per turn with coherence guards

## Description
Refresh player-view background each turn while enforcing coherence so visuals remain consistent with campaign style and current scene context.

## Acceptance Criteria
- [ ] Turn progression triggers player-view background update requests
- [ ] Coherence guards prevent drastic unintentional style/subject drift between turns
- [ ] Update cadence is performance-safe and can skip/regulate generation under load
- [ ] Last known good image persists when a turn update fails
