# M003.8 - Trust model, sandboxing, and content safety limits

## Description
Define safety boundaries for user-generated mod content and import execution.

## Acceptance Criteria
- [ ] Import accepts data files only; no executable/script payload support
- [ ] File size/count and parsing complexity limits are enforced
- [ ] Unsafe/blocked patterns are detected and reported before import
- [ ] Documentation defines trust expectations and local security caveats
