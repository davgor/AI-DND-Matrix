# m002.2 Host-run campaign service lifecycle

## Description
Implement host runtime lifecycle for multiplayer campaign sessions, including start, advertise, serve, and stop phases.

## Acceptance Criteria
- [ ] Host can start multiplayer mode for a selected campaign
- [ ] Host service lifecycle states are explicit (starting, online, degraded, stopping, offline)
- [ ] Host shutdown transitions campaign presence to offline cleanly
- [ ] Lifecycle state changes are observable by guest clients and sidebar status layer
