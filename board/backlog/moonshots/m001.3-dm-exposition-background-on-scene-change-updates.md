# m001.3 DM exposition background on scene-change updates

## Description
Update DM exposition background when scene context changes so staging visuals match narrative state.

## Acceptance Criteria
- [ ] Scene-change detection triggers DM background generation/update requests
- [ ] DM background updates are tied to scene context metadata and timestamped
- [ ] DM panel can continue functioning while background update is in-flight
- [ ] Failed background updates degrade gracefully to prior valid image
