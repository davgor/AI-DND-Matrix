# m005.10 Zero-hosting cost envelope and relay policy

## Description
Codify the economic constraint: developer does not pay for LLM inference or mandatory game hosting; optional relay (if any) has an explicit cheap/free envelope and a BYO-tunnel escape hatch.

## Acceptance Criteria
- [ ] MVP success path requires $0 developer LLM spend by design
- [ ] BYO tunnel path is documented as fully viable without any developer-operated service
- [ ] If a shared relay is proposed, monthly cost envelope, abuse controls, and shutoff plan are written
- [ ] “Free tier dies under load” failure mode has a user-visible fallback (switch to LAN/BYO tunnel)
- [ ] Go/no-go explicitly allows shipping without a shared relay
