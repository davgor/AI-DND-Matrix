# M003.2 - Parser and loader for text-based mod content

## Description
Build parser/loader that reads mod text files and transforms them into typed internal records.

## Acceptance Criteria
- [ ] Loader supports chosen source formats (for example markdown/yaml/jsonl) per spec
- [ ] Parsing errors return file path and line context when available
- [ ] Loader output is deterministic for identical inputs
- [ ] Unit tests cover malformed files, partial files, and mixed content types
