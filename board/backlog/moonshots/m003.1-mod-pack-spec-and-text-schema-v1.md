# M003.1 - Mod pack spec and text schema v1

## Description
Define the mod pack folder contract and text-file schema for homebrew content seeding.

## Acceptance Criteria
- [ ] Spec defines required folders/files and metadata manifest (name, id, version, author)
- [ ] Text schema supports spells, monsters, NPCs, gear, and extensible future types
- [ ] Schema includes taxonomy buckets, tags, and source/provenance fields
- [ ] Example mod pack included in docs with valid sample entries
