# M003.9 - Mod management UX: import, enable, disable, remove

## Description
Provide UI flow to manage installed mod packs without manual DB operations.

## Acceptance Criteria
- [ ] Settings or content manager view lists installed mods with status and version
- [ ] User can import new mod pack and see progress + actionable errors
- [ ] User can enable/disable mod pack per campaign or globally (as designed)
- [ ] Remove/uninstall flow handles linked records safely and predictably
