# m004.8 Grid exploration engine (move, collide, travel maps)

## Description
Implement engine-owned grid exploration: tile steps, wall collision, interaction targets, and multi-map transitions. No Electron/LLM imports; positions are validated before persist.

## Acceptance Criteria
- [ ] Player can attempt move N/E/S/W; engine rejects blocked tiles and stays put
- [ ] Stepping onto a transition moves the player to the linked map landing tile
- [ ] Adjacent interact selects the correct entity id for talk/combat provoke flows
- [ ] All movement/travel results are deterministic and unit-tested
- [ ] Import boundary: spatial rules live in `/engine` (or shared pure modules), not agents
