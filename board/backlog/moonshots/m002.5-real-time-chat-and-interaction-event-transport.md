# m002.5 Real-time chat and interaction event transport

## Description
Implement transport for multiplayer chat and gameplay interaction events between host and guests.

## Acceptance Criteria
- [ ] Guests can send/receive chat events in real time while host is online
- [ ] Interaction events are ordered and timestamped for deterministic replay/debugging
- [ ] Transport handles transient packet loss/disconnect with graceful retries
- [ ] Event transport failures do not corrupt authoritative host campaign state
