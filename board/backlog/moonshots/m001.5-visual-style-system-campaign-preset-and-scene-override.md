# m001.5 Visual style system (campaign preset + per-scene override)

## Description
Implement style controls combining campaign-level visual preset with optional per-scene overrides.

## Acceptance Criteria
- [ ] Campaign-level style preset is stored and applied by default
- [ ] Per-scene override is optional and composes safely with campaign preset
- [ ] Style metadata is persisted and replayable across sessions
- [ ] UI surfaces current style source (preset vs override) for transparency
