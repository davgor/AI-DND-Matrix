# m002.7 Multiplayer turn arbitration and conflict resolution

## Description
Define and enforce host-driven turn arbitration so simultaneous guest actions resolve deterministically.

## Acceptance Criteria
- [ ] Host applies deterministic ordering for competing guest actions
- [ ] Conflict outcomes are visible to participants with clear resolution feedback
- [ ] Turn-state transitions remain authoritative on host even under high interaction load
- [ ] Arbitration logic is covered by tests for representative conflict scenarios
