# M003.4 - Mod catalog schema and source attribution

## Description
Extend DB schema to store mod entries, source metadata, and ownership boundaries.

## Acceptance Criteria
- [ ] Tables/columns added for mod source id, version, author, and import timestamp
- [ ] Catalog records can be filtered by source type (core seed vs mod)
- [ ] Relations support spells, monsters, NPCs, gear, and cross references
- [ ] Migrations include rollback/forward safety and integrity checks
