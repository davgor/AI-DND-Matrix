# m001.2 Campaign-time region/location seed image generation

## Description
Generate initial visual seeds for key regions and locations during campaign creation to establish visual world identity.

## Acceptance Criteria
- [ ] Campaign creation flow triggers region/location image generation for defined seed targets
- [ ] Generated world assets are linked to campaign/region/location records
- [ ] Generation can run asynchronously without blocking core campaign creation completion
- [ ] Seeded images are available on first campaign load when generation succeeds
