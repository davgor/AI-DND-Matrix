# m002.8 Sidebar host online/offline status for multiplayer campaigns

## Description
Show multiplayer campaign availability in sidebar based on host online/offline state.

## Acceptance Criteria
- [ ] Sidebar marks multiplayer campaigns as online when host service is active
- [ ] Sidebar marks campaigns as offline when host is not currently running/open
- [ ] Status updates propagate without requiring full app restart
- [ ] Offline state provides clear join/observe limitations to guest users
