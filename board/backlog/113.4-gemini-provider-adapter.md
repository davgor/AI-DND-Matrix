# 113.4 Gemini provider adapter

## Description
Implement a Google Gemini API provider adapter behind `Provider.generate`, configurable by API key + model id from Settings.

## Acceptance Criteria
- [ ] `createGeminiProvider({ apiKey, model })` implements `Provider.generate`
- [ ] Honors `GenerateContext` (`systemPrompt`, `maxTokens`) with Gemini’s request shape
- [ ] Clear config/request errors; API key never appears in thrown messages or logs
- [ ] Unit tests with mocked fetch cover success and failure paths
- [ ] Settings “Test connection” works in Gemini mode
