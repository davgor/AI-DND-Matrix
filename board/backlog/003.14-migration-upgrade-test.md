# Schema migration upgrade test

## Description
Prove the forward-only migration system correctly upgrades an older save file when the app opens it.

## Acceptance Criteria
- [ ] A test creates a SQLite file at an earlier migration version (e.g. only migrations 1-2 applied)
- [ ] Opening that file with the full migration set applies all pending migrations automatically
- [ ] Data written under the old schema remains readable and correct after the migration runs
