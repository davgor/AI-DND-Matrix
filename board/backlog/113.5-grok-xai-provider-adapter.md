# 113.5 Grok (xAI) provider adapter

## Description
Implement an xAI Grok provider adapter (OpenAI-compatible chat endpoint or official xAI API) behind `Provider.generate`, configurable by API key + model id from Settings.

## Acceptance Criteria
- [ ] `createGrokProvider({ apiKey, model, baseUrl? })` implements `Provider.generate`
- [ ] Default base URL points at xAI’s public API; override only if explicitly needed
- [ ] Honors `GenerateContext`; truncation/error handling consistent with other cloud adapters
- [ ] Unit tests with mocked fetch cover success and failure paths
- [ ] Settings “Test connection” works in Grok mode
- [ ] Curated catalog seeds include at least one mid-tier and notes flagship options (e.g. grok-4.3 / grok-4.5 class ids current at implementation time)
