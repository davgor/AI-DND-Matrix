# oxlint strict config + governance

## Description
Configure oxlint with the agreed strictness: complexity, length, params, and nesting limits, plus a written governance note that relaxing rules requires explicit user permission.

## Acceptance Criteria
- [ ] oxlint config enforces cyclomatic complexity <= 10, max function length ~50 lines, max params <= 4, max nesting depth 3
- [ ] `npm run lint` script exists and runs clean against the current scaffold
- [ ] The config file contains a comment stating rule relaxations/overrides require explicit user permission before being made
