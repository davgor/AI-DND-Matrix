# Package/export script (Windows .exe)

## Description
Add the packaging script that produces a distributable Windows executable via electron-builder or electron-forge.

## Acceptance Criteria
- [ ] `npm run package` produces a `.exe` (or installer) in a build output directory
- [ ] The packaged app launches and shows the frameless window with titlebar from a clean checkout build
- [ ] Packaging is not wired into any CI workflow yet (local/manual only at this stage)
