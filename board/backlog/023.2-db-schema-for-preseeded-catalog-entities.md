# DB schema for preseeded catalog entities

## Description
Add database schema support for preseeded catalog content and taxonomy relationships.

## Acceptance Criteria
- [ ] Tables exist for creatures/enemies, spells, abilities, tags, and bucket mappings
- [ ] Schema supports provenance/source metadata and version tracking per seeded entry
- [ ] Indices support fast retrieval by bucket, level/difficulty, and archetype filters
- [ ] Migration tests verify install and upgrade behavior
