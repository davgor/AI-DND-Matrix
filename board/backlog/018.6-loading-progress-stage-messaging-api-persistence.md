## Description
Show meaningful loading status while generation API/provider call and persistence steps complete.

## Acceptance Criteria
- [ ] Loading UI displays stage/status text tied to real processing steps (for example request, parse, persist)
- [ ] Progress/status updates are deterministic and do not regress unexpectedly
- [ ] Messaging remains user-friendly while technical details stay in logs
- [ ] UI tests verify stage messaging across success and failure paths
