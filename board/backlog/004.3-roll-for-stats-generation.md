# Roll-for-stats score generation

## Description
Implement the 4d6-drop-lowest roll-for-stats generation method with a seedable RNG so it's deterministic in tests.

## Acceptance Criteria
- [ ] Rolling uses 4d6, drops the lowest die, sums the remaining three
- [ ] The RNG is injectable/seedable; the same seed always produces the same four scores
- [ ] Test confirms output scores fall within the valid 3-18 range per ability
