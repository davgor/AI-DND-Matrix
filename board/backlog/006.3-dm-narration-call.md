# DM agent: narration call wiring

## Description
Implement the DM agent's narration call: given the engine's resolution plus a freshly-pulled world-state snapshot, produce narration text.

## Acceptance Criteria
- [ ] The narration call's context includes region status, recent events, and story thread state pulled fresh from the DB at call time
- [ ] Unit tested that the narration call never receives a hand-invented pass/fail/damage value — only the engine's actual resolution result
