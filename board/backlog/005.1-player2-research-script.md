# Player2 research script

## Description
Write a standalone script (outside the app) that hits the running Player2 service at 127.0.0.1:4315 and logs the real request/response shape, since the API is currently unverified.

## Acceptance Criteria
- [ ] With Player2 running locally, the script successfully calls it and prints the response
- [ ] Findings (auth method, endpoint path(s), request payload shape, response payload shape, error format) are written down in a short reference doc or code comment
- [ ] The script is not part of the app's build/test pipeline (research-only artifact)
