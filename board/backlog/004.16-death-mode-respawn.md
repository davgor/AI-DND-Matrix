# Death mode execution: Respawn

## Description
Implement Respawn death-mode resolution: relocate, deduct cost, and enforce limits per the campaign's respawn_rules.

## Acceptance Criteria
- [ ] Given a lost dying sequence and `death_mode = respawn`, the character is relocated to `respawn_rules.location`
- [ ] The configured `respawn_rules.cost` is deducted from the character's currency
- [ ] A remaining-uses counter is decremented when `respawn_rules.limits` defines a limit; exhausting it falls back to Legendary behavior on the next death, unit tested
