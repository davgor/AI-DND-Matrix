# Currency debit/credit functions

## Description
Implement the engine-level currency debit/credit rules that the agent-facing shop interactions use.

## Acceptance Criteria
- [ ] `creditCurrency(character, amount)` and `debitCurrency(character, amount)` are pure functions returning new state + success/failure
- [ ] Debiting more than the current balance is rejected with an explicit insufficient-funds result, never silently clamped to zero
- [ ] Unit tested for a valid credit, a valid debit, and a rejected over-debit
