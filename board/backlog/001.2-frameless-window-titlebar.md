# Frameless window + custom titlebar

## Description
Replace the native window frame with a frameless Electron window and a custom React titlebar component matching the Discord-style chrome decision: draggable region, minimize/maximize/close buttons.

## Acceptance Criteria
- [ ] BrowserWindow is created with `frame: false`
- [ ] A `/renderer/titlebar` component renders a draggable bar (`-webkit-app-region: drag`) plus minimize/maximize/close buttons
- [ ] Clicking minimize/maximize/close calls the corresponding Electron window action via IPC and visibly works
- [ ] No native OS window frame/menu bar is visible when the app launches
