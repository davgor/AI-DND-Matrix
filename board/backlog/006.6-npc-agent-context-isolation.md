# NPC agent: context assembly + memory isolation

## Description
Implement the NPC agent's context assembly: only that NPC's own npc_memories plus world_facts tagged to its region/faction.

## Acceptance Criteria
- [ ] Context assembly for NPC A never includes NPC B's memory rows, even when both are seeded in the same test
- [ ] world_facts included are limited to those tagged to the NPC's region/faction
- [ ] Unit tested with seeded multi-NPC data
