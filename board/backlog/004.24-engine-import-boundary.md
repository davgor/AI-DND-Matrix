# Engine import-boundary test

## Description
Prove `/engine` has zero dependency on Electron, a DB client, or an LLM provider, since it must remain pure and independently testable.

## Acceptance Criteria
- [ ] A test or lint rule scans `/engine` source files and fails if any import references Electron, `better-sqlite3`, or an agent/provider module
- [ ] The check runs as part of `npm test` or `npm run lint` so it can't silently regress
