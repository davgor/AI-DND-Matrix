# Campaign generation: review/edit UI

## Description
Build the UI step where the player reviews and can edit the generated campaign summary before character creation.

## Acceptance Criteria
- [ ] After generation, a summary screen shows the generated regions/NPCs/story thread
- [ ] Player can edit text fields (e.g. region/NPC descriptions) before continuing; edits are persisted
- [ ] Continuing from this screen proceeds to character creation (ticket 009)
