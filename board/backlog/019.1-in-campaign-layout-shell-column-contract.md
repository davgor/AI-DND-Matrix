# In-campaign layout shell and column contract

## Description
Create a typed layout contract for in-campaign mode with a maximum four-column grid and deterministic region assignments.

## Acceptance Criteria
- [ ] Layout defines fixed semantic regions for campaigns, DM exposition, player interaction, and player sheet
- [ ] Desktop view supports up to four visible columns simultaneously
- [ ] Column width rules/minimums are defined to prevent overlap or clipping
- [ ] Unit/UI test verifies base shell renders all required regions
