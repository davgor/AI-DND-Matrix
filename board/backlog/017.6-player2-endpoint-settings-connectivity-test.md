# Player2 endpoint settings + connectivity test

## Description
Allow users to set a custom Player2 base URL and validate connectivity from Settings.

## Acceptance Criteria
- [ ] Settings include Player2 base URL input with sane default and clear format guidance
- [ ] URL validation catches malformed or unsupported endpoint formats before save
- [ ] "Test connection" action performs live connectivity check and returns success/failure result
- [ ] Connectivity failures present clear diagnostic message without crashing UI
