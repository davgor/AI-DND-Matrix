# Replace "new campaign" click behavior with modal entry

## Description
Change the main/side entry action so selecting "new campaign" opens a campaign-setup modal instead of immediately running generation.

## Acceptance Criteria
- [ ] Activating "new campaign" opens modal with no generation request sent yet
- [ ] Modal can be dismissed cleanly and returns user to prior screen state
- [ ] Existing campaign list/session state is unchanged until explicit submit
- [ ] UI test verifies entry, dismiss, and re-open behavior
