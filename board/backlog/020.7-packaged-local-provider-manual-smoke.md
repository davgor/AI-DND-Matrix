# Packaged local-provider manual smoke

## Description
Run a manual smoke check against a packaged build configured to force the local provider, confirming a real gameplay action succeeds end-to-end and that runtime unavailability fails gracefully and recoverably.

## Acceptance Criteria
- [ ] In a packaged build configured for the local provider only (no cloud fallback), a real generation call succeeds end-to-end for a representative gameplay action
- [ ] The smoke check is documented as manual, not part of CI
- [ ] Notes capture the exact packaged test setup: runtime availability, model location, config values used, and build version
- [ ] One expected-failure scenario (runtime missing/unhealthy) is also exercised, confirming a typed, user-actionable error and a stable, recoverable app
