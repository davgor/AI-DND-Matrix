# Create-campaign IPC wiring for generation counts

## Description
Thread `regionCount` and `npcsPerRegion` from renderer through preload-safe IPC into `createCampaignFromRequest` / `toSetupInput`, applying shared defaults when omitted.

Depends on **039.1**, **039.2**.

## Acceptance Criteria
- [ ] Preload + `window.campaigns.create` (or equivalent) accept the new fields on the typed payload
- [ ] Main process validates via `isValidCreateCampaignRequest` before invoking generation
- [ ] `CampaignSetupInput` (or generation entry type) receives resolved counts with defaults applied
- [ ] Unit tests verify IPC handler rejects invalid counts and passes valid counts to setup input
