# DC clamp integration between DM agent and engine

## Description
Wire the engine's DC clamp (ticket 004) into the path between the DM agent's proposed DC and the actual check resolution.

## Acceptance Criteria
- [ ] A DC proposed outside the sane range (e.g. 5-30) by a mock DM response is clamped before being used in check resolution
- [ ] Unit tested with a mock response proposing an out-of-range DC
