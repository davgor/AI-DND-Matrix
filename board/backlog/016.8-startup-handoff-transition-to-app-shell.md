# Startup handoff transition to app shell

## Description
Implement clean transition from loading screen to normal app shell once boot is complete.

## Acceptance Criteria
- [ ] When startup reaches ready, loading screen transitions once into app shell with no flicker/re-entry loop
- [ ] App shell interactive actions remain disabled until ready is confirmed
- [ ] Transition preserves selected route/start destination semantics
- [ ] Unit or integration test verifies single handoff behavior
