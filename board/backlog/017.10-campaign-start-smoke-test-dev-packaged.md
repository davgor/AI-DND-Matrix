# Campaign-start smoke test (dev + packaged)

## Description
Validate the new campaign modal + loading flow end-to-end in development and packaged app environments.

## Acceptance Criteria
- [ ] Dev-mode smoke verifies: open modal, fill form, submit, see loading states, complete handoff to review/edit
- [ ] Dev-mode expected-failure smoke verifies actionable error + retry/back-to-form behavior
- [ ] Packaged-mode smoke verifies the same primary success path
- [ ] Smoke runbook documents setup, steps, and outcomes
