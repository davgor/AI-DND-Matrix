# Column 2 scene refresh/update UX

## Description
Support DM exposition refresh/update behavior so scene-setting content can update as turns/actions resolve.

## Acceptance Criteria
- [ ] Column 2 updates when new narration/stage-setting content is produced
- [ ] Refresh/loading state is visible while waiting for updated exposition
- [ ] Failed exposition refresh displays actionable error without breaking other columns
- [ ] Update flow is tested for success and failure paths
