# 118.2 — D20 visual component (settled face)

## Description

Build a reusable renderer component for a stylized twenty-sided die that can display a face value `1–20`. This is the visual used during tumble frames and at settle — not the small randomize icon in campaign create (`FieldRandomDiceButton`).

Keep it CSS/SVG-friendly, themeable via play-view variables, and unit-testable for the displayed face / aria label.

## Acceptance criteria

- [ ] Component renders a recognizable D20 and shows the given face number clearly at settle size
- [ ] Accepts `face: number` (1–20) and optional size/className; invalid values handled safely (clamp or guard — documented)
- [ ] Accessible: `aria-hidden` while tumbling if decorative, or `role="img"` + label when result is meant to be announced
- [ ] Visuals use play-view tokens / CSS variables; no WebGL dependency
- [ ] Component test asserts the settled face text/label for a sample roll
