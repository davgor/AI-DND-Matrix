# Vitest test runner config

## Description
Wire up Vitest as the project's test runner so every later TDD ticket has a working `npm test` from day one.

## Acceptance Criteria
- [ ] Vitest config exists and supports TypeScript test files under `/src`
- [ ] `npm test` runs successfully with zero or placeholder tests present
- [ ] A trivial placeholder test (e.g. `1 + 1 === 2`) passes, proving the runner is wired correctly
