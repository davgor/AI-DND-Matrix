# 117.3 — Highlight new DM setting messages in Scene feed

## Description

When a new DM play-log entry with `sceneSetting: true` appears in the Scene feed (`filterDmExpositionEntries` / `renderFeedLine`), highlight **that feed line** (not the whole feed) with the shared glow for a few seconds. This is distinct from 117.2: the summary header may also glow if the setting updates the summary, and the new feed row should glow as the concrete message that arrived.

## Acceptance criteria

- [ ] A newly appended DM entry with `sceneSetting: true` receives the incoming-highlight class on its feed row
- [ ] Non-setting DM narration lines do **not** get this highlight (v1)
- [ ] Hydrated / already-present setting lines on mount do **not** glow
- [ ] If multiple setting lines exist, only the newly arrived id(s) glow — older setting lines stay unhighlighted
- [ ] Component test: append setting entry → that row highlighted; append normal DM line → no highlight
