# Delete-campaign smoke test (dev + packaged)

## Description
Validate end-to-end campaign deletion against a real campaign with a full data footprint (regions, NPCs with memories, characters with uploaded images, saves, events) in both dev and packaged builds.

## Acceptance Criteria
- [ ] Dev-mode smoke: generate a campaign, play enough to populate regions/NPCs/memories/saves/events and upload a character portrait, delete it, and confirm every related DB row and uploaded file is gone
- [ ] Dev-mode smoke confirms other existing campaigns and their data are untouched by the deletion
- [ ] Packaged-mode smoke confirms the same delete flow succeeds in a production build
- [ ] Smoke runbook documents steps and observed outcomes
