# Party-member agent: persistent relationship context

## Description
Implement the party-member agent's context assembly, including persistent relationship/history toward the player across the whole campaign (not just the current scene).

## Acceptance Criteria
- [ ] Context assembly includes relationship/history data spanning prior sessions, not just the current scene's recent events
- [ ] Unit tested with seeded history from an earlier "session" showing it's still included in a later call's context
