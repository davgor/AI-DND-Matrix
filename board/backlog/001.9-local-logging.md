# Local logging (no telemetry)

## Description
Wire up local-only file logging (e.g. `electron-log`) for errors/diagnostics, with no network-based telemetry or crash reporting anywhere in the app.

## Acceptance Criteria
- [ ] A logging utility writes to a local log file on disk (e.g. under app userData)
- [ ] At least one intentional error thrown during dev produces a corresponding log file entry
- [ ] No telemetry/crash-reporting network call exists anywhere in the codebase (verified by review/grep for any such SDK)
