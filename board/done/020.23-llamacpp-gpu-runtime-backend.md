# 020.23 — Acquire GPU-capable llama.cpp runtime (Vulkan default)

## Description

In-app “Acquire runtime” currently downloads the pinned Windows **CPU** zip (`win-cpu-x64`), so managed local inference stays on CPU even though lifecycle already passes `--n-gpu-layers all`. Switch the happy-path acquire to a **Vulkan** (GPU) build and let Settings pick CPU vs Vulkan before acquire.

Depends on 020.19 (discover/acquire). Extends epic 020 Local llama.cpp provider.
