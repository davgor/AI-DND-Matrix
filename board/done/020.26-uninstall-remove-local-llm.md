# 020.26 — Uninstall prompt to remove local LLM assets

## Description

Downloaded GGUF models and acquired `llama-server` runtimes live under Electron `userData/llamacpp/` and can be multi-GB. On Windows NSIS uninstall, ask whether to remove those local LLM assets (default **Yes**). Do **not** wipe campaigns, settings, or the rest of `userData` — only the `llamacpp` subtree. Skip the prompt during auto-update / reinstall (`isUpdated`).

Depends on the `userData` layout from 020.6.
