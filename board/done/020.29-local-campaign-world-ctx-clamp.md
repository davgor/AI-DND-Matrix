# 020.29 — Local campaign world gen: stop ctx/max_tokens hang

Campaign create on the local llama.cpp provider looks stuck on **World · step 3 of 9**, then fails with “The narrative engine returned an invalid campaign.” Root cause: bulk stages request `max_tokens: 8192` against the default `ctxSize: 8192`, so completions truncate or thrash; nested retries (3× stage × 5× outer seed) burn with no HTTP timeout and no logged cause.
